১। null এবং undefined এর মধ্যে পার্থক্য

undefined মানে ভ্যারিয়েবল ডিক্লেয়ার করা হয়েছে কিন্তু কোনো মান অ্যাসাইন করা হয়নি। এটি জাভাস্ক্রিপ্ট স্বয়ংক্রিয়ভাবে সেট করে।
আর null মানে ইচ্ছাকৃতভাবে "কোনো মান নেই" বোঝাতে ডেভেলপার সেট করেন।

undefined => default empty value
null => intentional empty value

২। map() এবং forEach() এর পার্থক্য

map() অ্যারের প্রতিটি উপাদানের উপর কাজ করে এবং একটি নতুন অ্যারে রিটার্ন করে।
forEach() শুধু প্রতিটি উপাদানের উপর কাজ করে, কিন্তু কিছু রিটার্ন করে না।

ডাটা রিটার্ন পেতে => map()
শুধু লুপ বা সাইড ইফেক্টের জন্য => forEach()

৩। == এবং === এর পার্থক্য

== শুধু মান তুলনা করে এবং প্রয়োজনে টাইপ কনভার্সন করে (type coercion)।
=== মান এবং টাইপ দুটোই তুলনা করে, কোনো টাইপ কনভার্সন করে না।

সবসময় === ব্যবহার করা নিরাপদ।

৪। API ফেচ করার ক্ষেত্রে async/await এর গুরুত্ব

async ফাংশনকে asynchronous করে Promise রিটার্ন করে।
await- Promise পূরণ হওয়া পর্যন্ত অপেক্ষা করে।
এটি কোডকে synchronous মতো করে এবং .then() চেইনিং এড়ায়।
Error handling সহজ হয়।


৫। JavaScript-এ Scope (Global, Function, Block)

Global Scope: কোন ব্লক বা ফাংশনের বাইরে ডিক্লেয়ার করা ভ্যারিয়েবল; পুরো প্রোগ্রামে অ্যাক্সেস করা যায়।
Function Scope: ফাংশনের ভিতরে ডিক্লেয়ার করা; শুধুমাত্র সেই ফাংশনের ভিতরেই অ্যাক্সেস করা যায়।
Block Scope: {} ব্লকের ভিতরে let ও const দিয়ে ডিক্লেয়ার করা; শুধু সেই ব্লকে অ্যাক্সেস করা যায়।

Scope নির্ধারণ করে একটি ভ্যারিয়েবল কোথায় ব্যবহার করা যাবে।
